Life Expectancy Prediction using Machine Learning

This project predicts life expectancy categories of countries using supervised machine learning algorithms.

Models Used

Logistic Regression

KNN

Naive Bayes

Decision Tree

Linear SVM

Evaluation Metrics

Accuracy, Precision, Recall, F1-Score, ROC-AUC, k-Fold CV

Result

Decision Tree performed best in predicting life expectancy categories.
